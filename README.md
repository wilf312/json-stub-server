# json-stub-server

## Install

```bash

# npm
$ npm install json-stub-server --save-dev

# yarn
$ yarn add json-stub-server --save-dev

```

## Usage

```js

var jsonStubServer = require('json-stub-server')

jsonStubServer()

```
